Typable

Typable is a third party iOS keyboard developed to improve smartphone use for the blind and visually impaired. The keyboard takes advantage of the 3D touch feature introduced with the iPhone 6s. Prompted by haptic and audio feedback, the user can navigated through letter selection as well as select other keyboard interactions with various touch gestures. Once downloaded, Typable can be used in any app that supports alternate keyboards.
